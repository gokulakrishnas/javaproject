Composer - Database Stocks Sample Project

This project queries one database table for a list of companies using a DBData block. It then accepts a company name as input using the DBInput block against the set of company names returned by the DBData block. After accepting a valid input, it queries another table in the same database for the latest stock quote for the company and speaks it out.

Setup Instructions

1. Open the /db/connection.properties file in the Connections Editor.

2.  Create a new connection - use the default name "ConnectionProfile1"

3.  Provide details for your database. Note: This application works only with SQLServer databases.

4.  Test the connection to make sure it works.

5.  Create the required tables in your database and populate them with sample data using scripts in db/Scripts folder. For SQLServer, use SQLServer.sql file. You will need to execute this script in the correct database using SQLServer Management Studio or another tool that can work with databases.
